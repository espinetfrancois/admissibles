Organisation générale du css:

  • layout.css : fichier pour l'organisation générale des éléments de la page
  • forms.css : formattage des formulaires
  • images.css : gestion de l'affichage image|text (gauche|droite ou droite|gauche)
  • menu.css : gestion du menu (organisation interne)
  • bonnes_adresses.css : page des bonnes adresses
  • text.css : formatage du texte (paragraphes, liste générales….)

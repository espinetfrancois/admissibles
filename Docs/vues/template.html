Organisation des templates : 

Les templates sont des bouts de code (html, php) inclus dans le layout (au moment des renders).

Ils sont tous dans le dossier 'templates'


Règles de rédaction : 

  • les fichiers contiennent le contenu (et non la div entourante) de la zone dont ils portent le nom (pour header, footer, menus…)
    • par exemple, footer.html ne contient que "footer-left", et non pas la div footer elle-même
  • les menus de l'application se trouvent dans le dans le sous-dossier 'menus' des templates.
    • Pour les ajouter, il suffit de faire $layout (instance de Layout.class.php)->addMenu('nomdufichier.extension)
    • Actuellement, cela supporte les fichiers php
